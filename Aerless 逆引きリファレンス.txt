Aerless 逆引きリファレンス (Quick Reference)
Aerlessでコードを書くためのクイックガイドです。

基本の読み書き
変数に値を保存したい is power = 100 （is で定義、再代入は power = 150）

文字や計算結果を表示したい do "Hello Aerless" （do は画面に出力する命令です）

文字列の中に変数を埋め込みたい do "現在のパワーは {power} です" （{ } で変数を展開します）

計算と判定
算術演算をしたい +, -, *, /（割り算）, %（余り）が使えます。

ランダムな数字が欲しい is dice = rand(1, 6)

条件によって処理を変えたい if hp < 20 { do "ピンチ！" } no { do "元気！" }

同じかどうか、大小を比べたい ==（等しい）, >（より大きい）, <（より小さい）

繰り返しと制御
条件を満たす間繰り返したい during energy > 0 { energy = energy - 1 }

ループの先頭に戻りたい go （Pythonの continue に相当します）

リスト（配列）を操る
リストを作りたい is items = ["草", "杖", "盾"]

特定の場所の中身を知りたい/変えたい at(items, 0) （取得） / set(items, 1, "光る杖") （更新）

要素を追加したい/数を数えたい push(items, "薬") / size(items)

関数の定義と連携
独自の関数を作りたい spell heal(target, amount) { return target + amount }

データを流れるように処理したい（パイプライン） 100 |> heal(20) |> do （左の結果を右の引数に渡します）

🛠️ システム・対話
ユーザーに数字や記号を打ってもらいたい is num = get_num("値を入力:") / is op = get_op("記号:")

キーが押されているか判定したい if key("z") { do "Zキーが押された！" }

画面を消したい / 処理を待ちたい cls() （クリア） / wait(1000) （1000ミリ秒待機）

別のファイルを読み込んで実行したい arts "library.als"
