# 🌌 Aerless

**Aerless（エアレス）** は、魔法を紡ぐような感覚で記述できる、Pythonベースの軽量スクリプト言語です。  

## ✨ 特徴

- **Smart REPL (Dialog Mode):** 起動するだけで対話スタート。
- **直感的なパイプライン:** `|>` 演算子を使用して、データをつなげて処理できます。
- **日本語親和性:** メッセージに日本語を取り入れ、開発をより身近なものにします。
- **ポータブル:** `aerless.py` ファイルひとつで動作。複雑な環境構築は不要です。

## 🚀 クイックスタート

### 起動方法
Pythonが入っている環境で、以下のコマンドを実行してください。

```bash
python aerless.py
